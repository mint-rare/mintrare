# MintRare
- NFT 거래소 BApp입니다.
- NFT 거래 수수료는 1%입니다.
- NFT 거래 시 판매자와 구매자 모두에게 거래 금액의 3%만큼 민트레어 토큰을 리워드로 지급합니다.
- 서비스에 대한 보다 상세한 내용은 [이 곳](https://www.notion.so/NFT-89774d8d16c34cbfa32d22c82869977a)에서 확인하실 수 있습니다.
- https://www.mintrare.org 에서 직접 서비스를 사용해 보실 수 있습니다 :)
